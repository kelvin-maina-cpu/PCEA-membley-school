# Downloads Directory

This directory should contain the following PDF files:

- `prospectus.pdf` - School prospectus
- `admission-form.pdf` - Admission application form
- `fee-structure.pdf` - Current fee structure
- `academic-calendar.pdf` - School calendar with important dates
- `cbc-curriculum.pdf` - CBC curriculum guide
- `school-rules.pdf` - School rules and regulations

## Instructions

1. Create or obtain these PDF documents
2. Place them in this `downloads/` directory
3. Ensure filenames match exactly as referenced in `index.html`
4. Test download links after uploading

## File Naming Convention

All files should use lowercase with hyphens:

- Correct: `school-prospectus.pdf`
- Incorrect: `School Prospectus.pdf`

## File Size Optimization

- Keep PDFs under 5MB each
- Use PDF version 1.5 or higher for better compression
- Consider using web-optimized PDFs for faster loading
